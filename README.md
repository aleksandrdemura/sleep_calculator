😴 Калькулятор сна – Multi‑Language Implementation
Тестовый репозиторий с реализацией консольного калькулятора сна на 7 языках программирования.
Приложение помогает рассчитать оптимальное время для пробуждения или засыпания на основе циклов сна (по 90 минут), а также вести дневник сна с аналитикой.

💤 Что такое Калькулятор сна?
Умный помощник для здорового сна. Он использует знание о циклах сна (каждый длится ~90 минут) и предлагает время пробуждения, которое придётся на конец цикла – это позволяет просыпаться бодрым, а не в глубокой фазе.

Основные возможности:
Расчёт времени пробуждения – введите время засыпания и количество циклов (по умолчанию 5), получите время для подъёма.

Расчёт времени засыпания – введите желаемое время пробуждения и количество циклов, программа скажет, когда лучше лечь спать.

Дневник сна – сохраняйте свои записи: дата, время засыпания, время пробуждения, продолжительность, оценка качества (1–5).

Статистика – средняя продолжительность и средняя оценка качества за все записи.

Управление записями – просмотр, удаление, сохранение и загрузка данных из файла (JSON).

✨ Особенности
Интерактивное меню с выбором действий.

Расчёт основан на циклах по 90 минут (пользователь может задать количество циклов от 1 до 10).

Автоматическая конвертация времени ввода/вывода в удобочитаемый формат.

Валидация времени (в формате HH:MM).

Цветной вывод (где поддерживается) для наглядности.

Сохранение записей в файл sleep_data.json (или эквивалент).

Загрузка данных при старте.
